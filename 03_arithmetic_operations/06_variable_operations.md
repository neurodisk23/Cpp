
As with other programming languages, C++ allows assignment operators.
The table below shows the most often used assignment operators supported by C++.



| Operator      | Example | Equivalent    |
| :---        |    :----:   |          ---: |

|+=|A+=B|A=A+B|
|-=|A-=B|A=A-B|
|* =|A* =B|A=A* B|
|/=|A/=B|A=A/B|
|%=|A%=B|A=A%B|
